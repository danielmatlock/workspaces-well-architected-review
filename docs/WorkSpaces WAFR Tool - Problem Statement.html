Amazon Web Services
Problem Statement

No Well-Architected Review Exists for Amazon WorkSpaces Fundamentals

Background

For Agentic Workspaces, there was no existing Well-Architected Framework Review that dived deep into the fundamentals of Amazon WorkSpaces — covering areas such as networking, directory services, image management, scaling, security, and cost optimisation specific to the service.

The Gap

TAMs had no structured, repeatable assessment to evaluate a customer's WorkSpaces deployment against best practices — and no automated way to generate a professional report from the findings.

Specific Challenges
  • No WorkSpaces-specific WAFR questions or review framework existed
  • Reports had to be manually created for each customer engagement — time-consuming and inconsistent
  • No standardised way to capture findings, score them, and present actionable recommendations
  • Each TAM reinvented the process independently, with no reusable tooling
Solution

Build a purpose-built WAFR tool for Amazon WorkSpaces that provides structured review questions across all pillars, automated scoring, and generates customer-ready reports — eliminating manual effort and ensuring consistent, high-quality output for every engagement.
